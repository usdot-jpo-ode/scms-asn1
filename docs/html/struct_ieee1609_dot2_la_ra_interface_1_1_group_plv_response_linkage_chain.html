Detailed Description

LA uses this data type to send group PLV linkage chain information to RA.

Parameters
jMaxdenotes the maximum count of linkage values to be sent per i.
otherLais the unique 16 bit ID of the second LA.
valuesis the outer sequence of size equal to the difference: iMax-iMin. It contains an inner sequence of encrypted PLVs of size jMax.
groupIdentifierdenotes the 32 bit string ID of a group.
