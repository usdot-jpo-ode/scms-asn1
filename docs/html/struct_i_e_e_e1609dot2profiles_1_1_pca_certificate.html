IEEE1609dot2profiles::PcaCertificate Struct Reference

Detailed Description

This data type defines a Pseudonym Certificate Authority's (PCA) certificate structure and its contents generated by the ICA. The PCA provides a public key (encryptionKey) so that other SCMS components can send data securely to this PCA.

Parameters
issuercontains the hash of the certificate contents.
toBeSignedcontains certificate content that is signed by the PCA.
