67 -- @brief The EndEntityRaInterfacePDU is the parent message type for messages
68 -- sent between end entities and the Registration Authority.
69 -- @class EndEntityRaInterfacePDU
70 -- @param eeRaCertRequest contains a message from an EE
71 -- to the RA to request a copy of
72 -- the RA’s currently valid
73 -- certificate.
74 -- @param raEeCertResponse is a message from the RA to the
75 -- EE as a response to
76 -- EeRaCertRequest.
77 -- @param eeRaPseudonymCertProvisioningRequest is a message from an EE to the
78 -- RA to request pseudonym
79 -- certificates.

81 -- EE to acknowledge the receipt
82 -- of EeRaPseudonymCertProvisioningRequest.
83 -- @param eeRaIdCertProvisioningRequest is a message from an EE to the
84 -- RA to request an identification
85 -- certificate.
86 -- @param raEeIdCertProvisioningAck is a message from the RA to the
87 -- EE to acknowledge the receipt
88 -- of EeRaIdCertProvisioningRequest.
89 -- @param eeRaAppCertProvisioningRequest is a message from an EE to the
90 -- RA to request an application
91 -- certificate.
92 -- @param raEeAppCertProvisioningAck is a message from the RA to the
93 -- EE to acknowledge the receipt
94 -- of EeRaAppCertProvisioningRequest.
95 -- @param eeRaAuthenticatedDownloadRequest is a message from an EE to the
96 -- RA to request the download of
97 -- certificates (once they are ready).
