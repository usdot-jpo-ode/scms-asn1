IEEE1609dot2ScmsProtocol::ScmsPDU Struct Reference

Detailed Description

The ScmsPDU is the parent structure that encompasses all parent structures of interfaces defined in the SCMS.

Parameters
versioncontains the current version of the data type. The version specified in this document is version 1, represented by the integer 1.
contentencloses the information of an SCMS interface.
ccmcontains the component certificate management interface structure.
eca-eecontains the interface structure defined for interaction between Enrollment Certificate Authority (ECA) and an End Entity (EE).
ee-macontains the interface structure defined for interaction between an End Entity (EE) and Misbehavior Authority (MA).
ee-racontains the interface structure defined for interaction between an End Entity (EE) and Registration Authority (RA).
la-macontains the interface structure defined for interaction between Linkage Authority (LA) and Misbehavior Authority (MA).
la-pcacontains the interface structure defined for interaction between Linkage Authority (LA) and Pseudonym Certificate Authority (PCA).
la-racontains the interface structure defined for interaction between Linkage Authority (LA) and Registration Authority (RA).
ma-pcacontains the interface st defined for interaction between Misbehavior Authority (MA) and Pseudonym Certificate Authority (PCA).
ma-racontains the interface structure defined for interactions between Misbehavior Authority (MA) and Registration Authority (RA).
pca-racontains the interface structure defined for interactions between Pseudonym Certificate Authority (PCA) and Registration Authority (RA).
ra-pgcontains the interface structure defined for interactions between Registration Authority (RA) and Policy Generator (PG).
