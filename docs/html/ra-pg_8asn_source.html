109 -- @brief RA uses this data type to request signed modified global parameters
110 -- inside a local policy file (LPF). An RA may generate an LPF with a

112 -- must be approved and signed by the PG before it can be distributed
113 -- to EEs.
114 -- @class RaPgPolicySignatureRequestMsg
115 -- @param version contains the current version of the data type. The version

119 -- @see Uint8
