Detailed Description

This data type is used by the LA to respond to a RaLaIndividualPreLinkageValueRequestMsg or a RaLaGroupPreLinkageValueRequestMsg with the requested PLVs.

Parameters
versioncontains the current version of the data type. The version specified in this document is version 1, represented by the integer 1.
requestHashis the has of the original request.
laIddenotes the unique hostname ID of the responding LA.
replycontains the requested PLVs if no errors occur, otherwise an appropriate error code is sent.
See Also
Uint8, HashedId8, LaHostnameId, ScopedLaRaError
LaRaPreLinkageValueResponseMsg ::= SEQUENCE {
version Uint8(1),
requestHash HashedId8, -- hash of the original request
laId LaHostnameId,
reply CHOICE {
success PreLinkageValueRequestResponse,
failure ScopedLaRaError
}
