Detailed Description

This data type contains local policy data generated by RA from global policy data derived from GPF of PG.

Parameters
temporalSeriesOfShuffleThresholdminimum # of individual cert requests before shuffle/send to PCA.
temporalSeriesOfCertsPerIPeriodcerts per i-period. overrides global value); default: 20
temporalSeriesOfLaOneHostLA1 256-bit unique hostname.
temporalSeriesOfLaTwoHostLA2 256-bit unique hostname.
temporalSeriesOfPcaHostPCA 256-bit unique hostname.
temporalSeriesOfRaX509TlsCertRA TLS certificate for connection over HTTP.
temporalSeriesOfLaX509TlsCertLA TLS certificate.
temporalSeriesOfPcaX509TlsCertPCA TLS certificate.
temporalSeriesOfSharedKeyUpdateIntervalmaximum time between changes to pre-linkage value enc/dec key.
