Detailed Description

This structure defines the response message to RaPcaCertRequestMsg. It is sent by PCA to RA.

Parameters
versionthe current version of the data type. The version specified in this document is version 1, represented by the integer 1.
requestHashcontains the hash of the original request, using the HashedId8 type as defined in 1609dot2-base-types.asn of [2]. In the case when the PCA cannot calculate the hash of the original request due to a hardError, this field shall be all zeroes.
replycontains the encrypted pseudonym certificates with corresponding linkage values for an individual OBE. A failure is sent if any form of error arises in communication between PCA and RA. Refer pca-ra-error.asn.
