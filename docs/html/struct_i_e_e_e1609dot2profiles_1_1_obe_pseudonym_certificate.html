Detailed Description

This data type defines an On-Board Equipment's (OBE) pseudonym certificate structure and its contents. The RA brokers the communication for provisioning this certificate from PCA to the OBE.

Parameters
issuercontains the hash of the certificate contents.
toBeSignedcontains certificate content that is signed by the OBE.
