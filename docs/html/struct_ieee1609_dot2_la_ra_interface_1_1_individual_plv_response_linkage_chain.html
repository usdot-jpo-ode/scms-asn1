Detailed Description

LA uses this data type to send individual PLV linkage chain information to RA.

Parameters
jMaxdenotes the maximum count of linkage values to be sent per i.
valuesis the outer sequence of size equal to the difference: iMax-iMin. It contains an inner sequence of encrypted PLVs of size jMax.
linkageChainIdis the LCI of the requested PLV.
