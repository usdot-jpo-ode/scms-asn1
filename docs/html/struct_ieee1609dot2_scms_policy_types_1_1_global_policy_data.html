Ieee1609dot2ScmsPolicyTypes::GlobalPolicyData Struct Reference

Detailed Description

This data type contains global policy data generated by PG.

Parameters
temporalSeriesOfScmsVersionSCMS Version, default value is 1
temporalSeriesOfCertChainFileIDFile ID number of the current GCCF
temporalSeriesOfOverdueCrlTolerancemax time to operate without a new CRL, specified in weeks (4 bytes)
temporalSeriesOfIPeriodi-value / i-period; default: 1 week
temporalSeriesOfMinCertsPerIPeriodminimum certs per i-period; default: 20
temporalSeriesOfCertValidityModelpseudonym cert validity model - "concurrent" or "non-concurrent"
temporalSeriesOfMaxAvailableCertSupplymax time covered by a certificate batch in years, default: 3 years
temporalSeriesOfMaxCertRequestAgemaximum time for individual cert request; to remain in aggregator; default: 2 days
temporalSeriesOfShuffleThresholdminimum # of individual cert requests before shuffle/send to PCA; default: 1000
temporalSeriesOfHashOfRequestSizebytes in "hash of request" between PCA and RA for individual cert requests; default: 32
temporalSeriesOfMaxGpfGccfRetrievalIntervalmaximum interval (in hours) before retreiving new GPF or GCCF; default: 1 hour
temporalSeriesOfRseApplicationCertValidityvalidity time for an RSE cert (in hours) Default value is 1 week + 1 hour = 168 hours
temporalSeriesOfRseApplicationCertOVerlapRSE application cert overlap; Default value is 1 hour
